## INICIANDO O PROJETO

Para instalar o projeto em sua maquina siga os passos abaixo.

### INSTALAÇÃO

1. Clone o repositório

   ```sh
   git clone https://github.com/NiloMarcos/RickAndMortyApp.git
   ```

2. Entre na pasta

   ```sh
   cd RickAndMortyApp
   ```

3. Instale as dependências

   ```sh
   yarn
   ```

   ou

   ```sh
   npm i
